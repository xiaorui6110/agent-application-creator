# xiaorui-ai-agent-platform

## 项目简介

基于 Spring Boot + Spring AI + LangChain4j 构建的 AI 智能体应用生成平台，支持用户输入自然语言需求自动生成可运行的代码文件（管理系统、小程序等），集成应用监控、AI 学习助手等核心功能。

### 核心技术栈

- 基础框架：Spring Boot 3.5.6
- 数据存储：MySQL 8.0、Redis 7.4、MyBatis Flex 1.8.0
- 安全组件：Spring Security、BCryptPasswordEncoder
- 其他：Lombok、Hutool
