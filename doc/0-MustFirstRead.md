## 项目介绍
本项目是基于 SpringBoot 3 + Spring AI Alibaba 独立开发的 AI 智能体应用开发平台，目的是扩展自主学习能力和理解新型 AI 智能体开发相关前沿技术。
>然后在选择了 Spring AI Alibaba 框架进行学习运用后，个人发现 Spring AI 相关框架目前对于智能体开发的支持相较于 LangChain4j 其实并不是特别完善。
所以，目前项目中有部分可优化的功能并未完全实现，比如智能体上下文工程、可观测性监控等，后续自己会随着框架不断更新而逐渐优化实现。

**项目的大体流程如下**：
1. 用户输入应用创建需求
2. 智能体根据需求生成应用代码
3. 用户可部署应用和下载应用代码
4. ......

**当然，目前项目还处于初期阶段，后续会不断完善和优化。**

## 项目模块
1. **用户模块**：登录、注册、个人中心
2. **智能体模块**：对话
3. **应用模块**：创建、部署、下载、监控
4. **对话历史模块**：保存、查询
> 这里就不做详细叙述了，会在后面的项目文档中详细记录自己的设计思路和实现过程。

## 项目计划
1. **MCP 模型上下文协议**：计划接入图片搜索 Image Search 和网络搜索 Web Search MCP 服务
2. **扩展应用模块**：计划接入评论系统，实现应用评论、点赞等功能，应用分类规则，应用排行榜等
3. **扩展用户模块**：增加些用户社交功能，实现用户关注、粉丝、好友等功能等

> 以上是后续可以比较容易实现的功能，会根据自己的实际安排去做实现。
