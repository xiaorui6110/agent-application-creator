# P3-2 应用模板化

这一阶段不做复杂的模板平台，而是先落地一个本地可用、可沉淀优秀应用代码的轻量模板机制。

## 1. 能力目标

- 支持从现有应用创建模板
- 支持查看本地模板列表
- 支持基于模板快速创建新应用
- 模板元数据落库，模板代码文件保存在本地目录

## 2. 存储设计

- 数据表使用 `xr_app_template`
- 模板目录使用 `app.storage.code-template-dir`
- 默认路径为 `./tmp/code_template`
- 每个模板目录包含模板代码文件
- 模板目录结构为 `{templateId}/files/`

## 3. 对外接口

- `GET /app/template/list`
- `POST /app/template/create`
- `POST /app/template/use`

## 4. 当前约束

- 当前模板能力以本地模板沉淀为主
- 暂不支持模板在线分享和审核
- 暂不支持模板市场、模板搜索和模板版本管理
- 当前模板删除和编辑能力尚未补齐

## 5. 后续扩展

- 可以继续向以下方向扩展
- 模板标签
- 模板导入导出
- 模板预览图和说明页
- 与社区共享能力联动
