# 项目初始化模块设计

> xiaorui

## 总览
基于 Spring Boot 构建的标准化项目初始化骨架，已完成企业级开发必备的基础能力搭建，包括统一依赖管理、全局异常处理、标准化返回结果、参数校验、日志配置、Redis / 数据库集成、工具类封装等核心初始化功能，可直接作为业务项目的底层基座，开箱即用。

## 功能模块

### 1. 统一依赖管理
- 使用 Maven 管理项目依赖，确保依赖版本统一且无冲突。
- 提供常用依赖的 POM 文件，便于快速引入。

### 2. 全局异常处理
- 统一捕获并处理业务异常，返回标准化的错误信息。
- 支持自定义异常类型，便于业务逻辑区分。

### 3. 标准化返回结果
- 定义统一的 API 返回格式，包含状态码、消息和数据。
- 支持分页查询结果的标准化封装。

### 4. 日志配置
- 集成 Logback 日志框架，支持多环境日志级别配置。
- 提供日志切面，自动记录关键业务操作日志。

### 5. Redis / 数据库集成
- 默认集成 Redis 缓存，支持分布式锁功能。
- 提供 MyBatis-Flex 配置，简化数据库操作。

### 6. 工具类封装
- 提供常用工具类，如日期处理、字符串处理、加密解密等。
- 支持自定义工具类扩展。

## 项目目录结构

```plaintext
src/main/
├── java/com/xiaorui/
│   ├── XiaoruiSpringbootInitApplication.java  # 启动类
│   ├── common/                                # 公共模块
│   ├── config/                                # 配置类
|   ├── constants/                             # 常量类
│   ├── controller/                            # 控制器层
|   ├── enums/                                 # 枚举类
│   ├── exception/                             # 异常处理
│   ├── generator/                             # 数据访问层代码生成工具
│   ├── model/                                 # 数据模型
│   └── util/                                  # 工具类
└── resources/
├── application.yml                        # 主配置文件（环境选择）
├── application-dev.yml                    # 开发环境配置
├── logback-spring.xml                     # 日志配置（暂未配置）
└── mapper/                                # MyBatis Flex Mapper文件
```

## 使用说明
1. 克隆项目到本地。
2. 根据业务需求修改配置文件（如 `application-dev.yml`）。
3. 启动项目，验证初始化功能是否正常。
4. 基于此骨架开发业务模块。
