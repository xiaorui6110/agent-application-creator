# 用户模块设计文档

> xiaorui

## 设计思路

### 1. 注册登录

- 通过邮箱进行注册登录，保证平台安全性
- 登录信息token存储在redis中
- 支持多端登录（Web、移动端）
- 登录超时自动刷新token

### 2. 密码加密

- 采用BCrypt加密，更安全
- 后续会将前端传输过程中加密
- 密码强度校验（至少8位，包含大小写字母、数字和特殊字符）

### 3. 权限校验

- AOP注解的方法鉴权
- 基于角色的访问控制（RBAC）
- 支持动态权限配置

### 4. 用户信息管理

- 支持用户基本信息的修改
- 用户行为日志记录

### 5. 安全措施

- 登录失败次数限制（防止暴力破解）
- 敏感操作二次验证（如修改密码）

### 6. 扩展功能（待实现）

- 用户分组（团队协作场景）
- 用户标签（自定义分类）
- 用户数据导出（支持CSV格式）
