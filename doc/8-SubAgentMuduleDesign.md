# 副 Agent 模块设计文档
> xiaorui，新增一个平台代码优化的副 agent。

## 设计思路
考虑到如何将 agent 的应用代码生成能够更加符合平台的风格等，
所以新增一个副 agent 来优化应用代码生成，并将优化建立反馈给主代码生成 agent。

## 流程
主要是在主 agent 执行完成之后，收集主 agent 生成的代码结果，
异步执行副 agent，然后收集副 agent 的执行结果，
在下一次主 agent 执行时将信息拼接到 message 中。

> 大体的调用流程同主 agent，主要还是考虑多智能体的协作方式。
